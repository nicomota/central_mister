Integração com a SIEG

Busca automática de notas fiscais dos clientes

Resumo

Descubra como ativar a integração com a Sieg no Mister Contador de forma simples e rápida. Este artigo irá guiá-lo passo a passo para habilitar a chave API para o Mister realizar a busca automática das NF-e, NFC-e, CTe e NFS-e dos seus clientes, no Cofre da Sieg.

Introdução

Está pronto para simplificar ainda mais sua contabilidade? Ativar a integração com a Sieg no Mister Contador é essencial para uma gestão eficiente das suas notas fiscais. Neste artigo, vamos mostrar como realizar esse processo de forma descomplicada e ágil.

Visão Geral do Recurso/Funcionalidade

A integração com a Sieg permite que você sincronize as notas fiscais dos seus clientes diretamente no Mister Contador. Para ativar essa funcionalidade, é necessário habilitar a chave API da Sieg e configurar a integração das empresas na página do Mister Contador.

Passo a Passo

  1. Acesse o site da Sieg com o e-mail de administrador e vá para "Minha Conta" > "Integração Chave API".
  2. Habilite a chave API e copie o código.
  3. Envie o código da chave API para o chat de atendimento na página do Mister Contador para decodificação.
  4. Na página do Mister Contador, acesse "Contabilidade" no menu lateral.
  5. Habilitar a integração de notas fiscais, informando o e-mail do administrador que foi utilizado para acessar a Sieg e a chave API, que deve ser decodificada pelo time de atendimento do Mister.
  6. No menu do cliente, clique em "NF-e" e depois clique na engrenagem (imagem da engrenagem) e preencha os campos de "data de início (o Mister irá buscar de forma automática, a partir do mês anterior ao cadastro da empresa)" e em seguida em marque a opção "habilitar integração". Aqui serão carregadas as NF-e, NFCe e CTe.
  7. Para as notas de serviços, no menu do cliente, clique em "NFS-e" e depois clique na engrenagem (imagem da engrenagem) e preencha os campos de "data de início (o Mister irá buscar de forma automática, a partir do mês anterior ao cadastro da empresa)" e em seguida em marque a opção "habilitar integração".

Melhores Práticas e Dicas Avançadas

  • Certifique-se de informar corretamente o e-mail com perfil de "administrador" na Sieg e a chave API correspondente ao e-mail, decodificada, para garantir a integração bem-sucedida.
  • Lembre-se de habilitar a integração das NF-e e das NFS-e, em todas as empresas que você cadastrar no Mister Contador.
  • O Mister buscará automaticamente as notas a partir do mês anterior à data de cadastro da empresa. Se o usuário desejar, poderá carregar manualmente, quantos períodos de notas forem necessários.

Perguntas Frequentes (FAQ)

  • Onde encontro a chave API da Sieg?
    A chave está disponível no menu "Integrações" no site da Sieg, acessível apenas pelo e-mail administrador.
  • Preciso decodificar a chave API antes de habilitar a integração no Mister Contador?
    Sim, a chave API precisa ser decodificada para garantir a integração correta.
  • O que o Mister irá buscar no cofre da Sieg?
    Notas NF-e, NFC-e, CTe e NFS-e.
  • Quando ocorre a busca de notas?
    O Mister fará a busca das notas no cofre da Sieg, nas sextas-feiras, a partir das 20:00, até a segunda-feira pela manhã, a partir do período anterior a data de cadastro da empresa.
  • Posso carregar manualmente quantos períodos de notas no Mister?O usuário poderá carregar manualmente, quantos períodos de notas desejar.

Exemplos de Uso

João, colaborador do departamento contábil, acabou de cadastrar uma novam empresa e habilitou a integração com a Sieg. Na próxima segunda-feira, o Mister irá mostrar as notas desse cliente, no item NF-e da empresa.

Seguindo os passos simples fornecidos neste artigo, o Mister irá realizar a busca automáticas das notas fiscais dos seus clientes, direto do cofre da Sieg.
