Entendendo o Registro Contábil

Registro contábil/Conciliação de lançamentos

Resumo

Após o upload dos insumos (xml das notas, comprovantes de pagamento e extratos bancários) no Mister Contador, o próximo passo é entender como será a contabilização no Registro Contábil. Nesta etapa, é onde o usuário irá realizar a contabilização da movimentação bancária das empresas. Neste artigo, vamos explorar de maneira simples e direta como realizar essa tarefa de forma eficiente.

Introdução

O Mister Contador irá realizar o lançamento contábil de forma automática nas contas contábeis de fornecedores, quando identificar o cnpj do beneficiário/fornecedor, nos comprovantes de pagamento, na NF-e de entrada e na conta contábil do fornecedor, no plano de contas. Quando não ocorrer o lançamento automático, o usuário precisará informar ao Mister Contador, as contas contábeis corretas para realizar a contabilização, associando a conta contábil, ou criando regras. Entender o processo de Registro Contábil é essencial. Também, é a garantia de que toda movimentação bancária dos seus clientes serão devidamente lançadas de forma precisa nas contas contábeis corretas.

Visão Geral do Recurso/Funcionalidade

Na tela de Registro Contábil do Mister, é mostrado o "espelho" do extrato, ou da planilha que o usuário utilizou para substituir o extrato bancário. Por exemplo: quando for um pagamento de fornecedor, no extrato bancário ou na planilha, o valor será um débito. E no registro contábil, irá aparecer como crédito na conta contábil do banco, para ser lançado a débito na conta contábil do fornecedor.

Ao acessar a aba lateral do Mister, você encontrará a opção "Registro Contábil". Aqui, você verá uma interface com três informações:

Registro Contábil:

  • Linhas azuladas - representam os lançamentos da conta contábil do banco cadastrado;
  • Linhas brancas - indicam os lançamentos conciliados automaticamente pelo sistema;
  • Linhas amarelas - destacam os lançamentos que requerem a intervenção do usuário para indicar a conta contábil correta, que poderá ser feita de 03 maneiras:
    • Através da associação da conta;
    • Associar ou criar uma regra;
    • Por meio de uma edição avançada.

Passo a Passo

1. Acessar o Registro Contábil:

Após acessar as informações da empresa (clicar em visualizar), acesse a aba "Registro Contábil" no Mister.

2. Identificar os tipos de lançamentos:

Identifique os lançamentos das linhas azuladas, linhas brancas e as linhas amarelas.

3. Conciliar lançamentos das linhas amarelas:

Para os lançamentos das linhas amarelas, realize a conciliação manualmente:

Ao clicar na linha específica do histórico, o usuário poderá criar uma regra, definindo um dos critério disponíveis (histórico bancário, nome do beneficiário ou usando uma informação extra) quando possível. Vale lembrar que será necessário estabelecer uma conta contábil e seu respectivo histórico contábil, por fim essa funcionalidade irá aplicar essa regra para todos os casos semelhantes.

Ps. Ideal para ser utilizado quando o comprovante de pagamento correspondente estiver ausente (por qualquer motivo) na plataforma.

Ao clicar na linha específica do histórico, o usuário também poderá indicar diretamente uma conta contábil específica para esse lançamento contábil. Vale lembrar que nesse caso a indicação feita vale exclusivamente para esse registro, não gerando uma recorrência para esse caso de uso.

O usuário poderá também usar um recurso mais sofisticado chamado de edição avançada. Clicando no "lápis", o usuário perceberá nessa funcionalidade a possibilidade de criar lançamentos em lote. Ideal para o registro de juros, descontos, tarifas e etc.

Melhores Práticas e Dicas Avançadas

  • Utilize a função de criação de regras para automatizar a conciliação de lançamentos recorrentes.
  • Esteja atento aos lançamentos das linhas amarelas, pois são eles que necessitam de intervenção manual.
  • A "edição avançada" é uma ferramenta poderosa para ajustes precisos nos seus registros contábeis. Pois é nela que será possível realizar os lançamentos múltiplos.

Perguntas Frequentes (FAQ)

    2. Você precisa conciliar os lançamentos das linhas amarelas, através de "associar conta", "associar regras"; ou "edição avançada".

Exemplos de Uso

  • João precisa conciliar os lançamentos do extrato bancário de uma empresa cliente, no Mister. Para os lançamentos que não forem conciliados de forma automática, ele poderá utilizar as regras de conciliação para automatizar o processo para os próximos períodos.
